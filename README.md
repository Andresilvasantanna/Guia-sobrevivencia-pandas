🧠 O que é o Pandas?

O Pandas é uma biblioteca open-source construída sobre o NumPy, amplamente usada em ciência de dados, engenharia de dados e análise estatística.
Ela permite trabalhar com dados de forma semelhante a planilhas (Excel) ou tabelas de banco de dados, mas com muito mais flexibilidade e poder.
